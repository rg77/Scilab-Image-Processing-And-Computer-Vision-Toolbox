FOSSEE Image Processing Toolbox

  • FOSSEE Image Processing Toolbox
    • colorChange — This function seamlessly mixes two differently colored versions of the input image.
    • convert — This function converts a matrix to another data type with optional scaling.
    • detailEnhance — This filter enhances the details of a the given image.
    • edgePreservingFilter — This function smoothens the given input image while preserving the edges.
    • getDepth — This function returns the depth of the input matrix element.
    • getStructuringElement — This function returns a structuring element required for the morphological operations.
    • illuminationChange — This function is used to reduce specular reflections or highlight under-exposed foreground objects.
    • localMaximaFinder — This function returns local maxima in input matrice.
    • minAreaRect — This function is used to find a rotated rectangle of the minimum area enclosing the input 2D point set.
    • minEnclosingCircle — This function is used to find the circumcircle of an object constituted by the given set of input points.
    • partition — This function is used to split an imageSet.
    • pencilSketch — This function is used to give a non-photorealistic pencil-like line drawing effect to the given input image.
    • pyramid — This function computes the pyramid reduction or expansion of input_image.
    • regionFill — This function restores the selected region in an image using the region neighborhood.
    • rgb2gray — Given an original color image, this function converts it into a single channel grayscale image and return the same.
    • roiFreeHand — This is a gui based function which is used to create a mask by selecting the region of interest
    • roiPoly — This function creates mask by selecting the ROI as a poygon.
    • seamlessClone — This function is used to seamlessly clone a source image onto a destination image.
    • stylization — This function aims to produce non-photorealistic digital imagery with a wide variety of effects.
    • templateMatcher — This function compares a template in overlapped image regions and returns the coordinates of the point of match.
    • textureFlattening — This function washes out the texture of the selected region in the input image, giving its contents a flat aspect.
    • watershed — This function returns a marker based segmented image for the given input image.
